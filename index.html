☁️ Sincronização GitHub

Salva os dados automaticamente no repositório GitHub após cada importação. Na próxima abertura do site, os dados carregam sozinhos em qualquer dispositivo.

Token de acesso
Personal Access Token com permissão de escrita no repositório
Como criar o token:
1. Acesse github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Clique em Generate new token
3. Em Repository access, selecione apenas silveira-bi
4. Em Permissions → Contents, selecione Read and Write
5. Gere e cole o token acima
